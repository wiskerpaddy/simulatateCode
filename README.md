# 🎹 Chord Flick JS
直感的なフリック操作で本物の楽器音を奏でる、Webベースのコード・シミュレーターです。
面倒なキーボード操作なしで、スマホやPCから「ジャカジャカ」とコード演奏を楽しめます。

## 🌟 主な機能
- **IMEスタイル・フリック操作**:
- **タップ: Majorコード**
- **上: minor**
- **右: 7th**
- **下: m7**
- **左上: M7**
- **右上: sus4**
- **本格的なサンプラー音源**:
Tone.jsのサンプラー機能を活用し、実際のピアノやアコースティックギターの録音データを再生。

- **ダイナミック・ディスプレイ**:
弾いているコード名をリアルタイムで表示するデジタルサイバー風パネル。

- **インテリジェント鍵盤表示**:
発音される音に合わせて、画面上の鍵盤が1オクターブ下で光り、自動スクロールで追従。

- **ギター・ストローク・シミュレーション**:
ギター選択時、各音をわずかに（0.04秒）ずらして発音。弦をかき鳴らすニュアンスを再現。

## 🛠 使用技術
- **Tone.js**: 音響合成・サンプラー制御
- **JavaScript (ES6+)**: ロジック・フリック判定
- **CSS3**: フレックスボックス、アニメーション、サイバーパンク・ディスプレイUI
- **HTML5**: セマンティック・マークアップ

## 🚀 セットアップ
本リポジトリをクローンまたはダウンロードします。
index.html をブラウザで開きます。
注意: サンプラー音源を外部から読み込むため、コンソールに Piano Ready や Guitar Ready と表示されるまで数秒待機してください。
